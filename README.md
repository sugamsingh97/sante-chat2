# SantéChat

Application d'analyse de croquettes pour chats.

## Prérequis

- Node.js
- npm

## Installation

```bash
npm install
```

## Lancement de l'application

### Mode développement
```bash
npm run dev
```

### Mode production
```bash
npm start
```

Dans les deux cas, l'application sera accessible sur http://localhost:5000
